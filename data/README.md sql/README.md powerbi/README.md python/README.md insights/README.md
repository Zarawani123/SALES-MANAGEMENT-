# Sales Performance Analysis

## Objective
Analyze sales and profitability data to uncover performance trends and provide actionable business insights.

## Business Questions
- Which regions generate high revenue but low profit?
- Which product categories contribute to losses?
- How do customer segments impact revenue and margin?
- What actions can improve profitability?

## Tools Used
- Excel (data cleaning & pivot analysis)
- SQL (aggregations, joins, KPI queries)
- Power BI (dashboard visualization)
- Python (exploratory data analysis)
